# C-Programs
A compilation of C programs focused on foundational programming concepts, control structures, functions, and logical problem solving.

## About This Repository

This repository contains fundamental and intermediate C programs covering core programming concepts. It is designed for beginners, first-year engineering students, and anyone building a strong foundation in C programming.

The programs are organized from basic concepts to more advanced topics including arrays, strings, functions, recursion, pointers, and structures. The repository will continue to grow with additional programs and concepts over time.

## How to Run the Programs

1. Clone the repository:

   ```bash
   git clone <your-repository-link>
   ```

2. Navigate into the folder:

   ```bash
   cd C-Programming
   ```

3. Compile the program:

   ```bash
   gcc filename.c -o output
   ```

4. Run the program:

   ```bash
   ./output
   ```
